# AndroidAGC
[![GPLv2](https://img.shields.io/badge/license-GPL-blue.svg)](https://github.com/Thymo-/AndroidAGC/blob/master/LICENSE)

AndroidAGC is a DSKY peripheral simulation using Ron Burkey's yaAGC on Android.

## Requirements
- Android Studio
- Android SDK 27
- Android NDK
- Cmake
- LLDB (Optional)
